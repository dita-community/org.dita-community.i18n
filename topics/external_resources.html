External Resources

This project draws on a number of resources:

CEDICT Project

http://www.mdbg.net/chindict

The CC-CEDICT project is an open-source Simplified Chinese dictionary reflecting about 140,000 words. It is the basis for the dictionary used in the i18n library's Simplified Chinese collator.

Hanban HSK exam Preparation Resources (HSK东西)

HSK东西

This site provides resources that support studying for the Hanban HSK (Hanyu Shuiping Kaoshi) exam, a free Chinese proficiency exam for Chinese language learners. The i18n project uses word lists and sample sentences from the hskhsk project as source data for exercising the zh-CN collation and other processing.
